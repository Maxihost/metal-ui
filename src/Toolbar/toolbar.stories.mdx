# Available components

Metal exports the following components for the Toolbar:

- Toolbar
- ToolbarButton
- ToolbarLink

# API

This component uses the Radix UI `Toolbar` primitive. Refer to [this page](https://radix-ui.com/primitives/docs/components/toolbar) for the API reference, but note components in Metal UI do not use dot-notation.

There's an additional `active` prop available for selectable components.

# With DropdownMenu

You can render the DropdownMenu as part of the toolbar

<Preview>
  <Story name="With DropdownMenu">
    <Toolbar>
      <ToolbarButton>Button 1</ToolbarButton>
      <ToolbarButton>Button 2</ToolbarButton>
      <Dropdown>
        <DropdownTrigger as={ToolbarButton}>I am a Dropdown</DropdownTrigger>
        <DropdownContent>
          <DropdownItem>Recently updated</DropdownItem>
          <DropdownItem>Least recently updated</DropdownItem>
        </DropdownContent>
      </Dropdown>
    </Toolbar>
  </Story>
</Preview>
